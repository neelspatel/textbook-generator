Summalyze

Organizing Information to Aid Learning

Take our project for a spin!

What is it?

Summalyze is a new way to learn about topics. It's a search engine to return not just links, but knowledge.


When you search for biology, you'll get an organized set of information with all the information you need to learn about biology. The information is sorted in order of complexity and also natural order of prerequisites.


Even for more complex topics like manifolds, Summalyze returns a set of information in order from the basics you need to know to in depth articles on specific topics.

How does it work?

  1. Calculate keywords. We implemented tf-idf scores to calculate the important words for any given query. We then parse, process, and clean this list of keywords.
  2. Search for spanning combinations. We find search results for spanning unique combinations of these keywords, and then get the unique set of URLs for these combinations.
  3. Process text from each URL. Get the source HTML, clean any markup, tokenize into words, and calculate the frequency distribution of each word.
  4. Calculate complexity and natural prerequisite order. Our algorithms calculate order based on various keyword frequency distributions.
